Магістерська робота

Розробка системи для моніторингу за комп'ютерним класом

Виконав: студент 641м групи Червеняк І. І.

Керівник: к.ф.- м.н., доцент Антонюк С.В.

Основною ціллю даної магістерської роботи є проектування та розробка системи із використанням веб-технологій для аналізу та відстеження стану ПК, а саме:

  1. Створити базу даних для збереження та доступу до даних про комп'ютери.
  2. Створити сервер, який буде підключатися до бази даних; виводити дані з бази даних на веб-сторінку; добавляти, видаляти та редагувати інформацію з бази даних.
  3. Створити клієнтську програму, яка буде передавати дані про стан компютера на сервер.
  4. Створити веб-сторінку, для відображення інформації з бази даних та взяємодії з нею.
  5. Переконатися, що:
    • дані, які відображає веб-сторінка є актульними;
    • інформація про компютер є актуальною;
